# User Flows

## Flow 1: Dashboard → Insight
Dashboard
→ tap "Подробнее" in AI card
→ AI Insight Detail
→ Back

## Flow 2: Dashboard → Category drilldown
Dashboard
→ tap category row (e.g., Кафе и рестораны)
→ Category Detail (transactions + totals)
→ Back

## Flow 3: History + filters
History
→ set segment: Все/Расходы/Доходы
→ open Filters (top-right)
→ apply date/category/source filters
→ open transaction row
→ Transaction Detail

## Flow 4: Add transaction
Tap +
→ choose type (Расход/Доход)
→ enter amount/date/time/source/note/category
→ Save
→ success feedback
→ appears in History/Dashboard

## Flow 5: Import statement
Import tab
→ select PDF/image
→ parsed preview list
→ edit categories per row
→ Save X transactions
→ success + reflected in Dashboard/History

## Flow 6: Settings
Settings
→ Account
→ Notifications
→ Categories
→ Cards/Accounts
→ Export Data
→ optional Erase all data confirmation
